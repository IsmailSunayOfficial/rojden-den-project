# ❤️ Честит рожден ден, любов моя

Романтичен, интерактивен сайт-подарък, разработен с React + Vite + TypeScript + Tailwind CSS + Framer Motion.

## 📁 Структура на проекта

```
src/
├── components/       → всеки екран/секция е отделен файл
├── data/             → текстове, снимки (пътища) и езици — редактираш САМО тук
├── hooks/            → малки помощни функции (брояч, typewriter, YouTube плейър)
├── App.tsx           → събира всички секции в правилния ред
└── index.css         → цветове, шрифтове, глобални стилове
```

Ако искаш да смениш **текст**, **снимка** или **дата** — първо провери в `src/data/`.
Почти никога няма да се налага да пипаш логиката в `components/`.

## 🖼️ Как да добавиш истинските снимки

1. Сложи снимките си в `src/assets/images/` (напр. `hero.jpg`, `purva-sreshta.jpg`, `gallery-1.jpg`...).
2. Отвори съответния data файл:
   - Заглавна снимка → `src/components/HeroSection.tsx` (константата `HERO_IMAGE`, по подразбиране `null`)
   - Снимки в историята → `src/data/timelineData.ts`
   - Снимки в галерията → `src/data/galleryData.ts`
3. Добави `import` в началото на файла и замени `null`/placeholder линка, напр.:

```ts
import heroPhoto from '../assets/images/hero.jpg'
// ...
const HERO_IMAGE = heroPhoto
```

Докато `HERO_IMAGE` е `null`, началният екран показва тих градиент вместо снимка (без никакъв текст върху него) — веднага щом добавиш снимката, тя ще се появи на нейно място.

## 🎞️ Как да добавиш GIF-ове за мечтите/моментите

Секция "Нямам търпение да преживеем още толкова красиви моменти" показва по един GIF за всяка мечта. Докато не добавиш GIF, се вижда изящен placeholder с емоджито и точното име на файла, което очаква.

1. Сложи GIF файловете в `src/assets/gifs/` — очакваните имена (по ID) са:
   - `zaedno.gif` → Да живеем заедно
   - `rojden-den.gif` → Да празнуваме рождения ден заедно
   - `kapadokia.gif` → Кападокия
   - `ryka-za-ryka.gif` → Ръка за ръка
   - `pochivki.gif` → Почивки
   - `semeystvo.gif` → Семейство
2. Отвори `src/data/futureData.ts`, добави import за всеки GIF отгоре, напр.:

```ts
import zaednoGif from '../assets/gifs/zaedno.gif'
```

3. Замени `gif: null` с `gif: zaednoGif` (и аналогично за останалите).

## 🌍 Секцията "Обичам те" на 100 езика

Вместо автоматично сменящи се карти, сега можеш сам да избереш език — с флагче — и да видиш как звучи "Обичам те". Има и търсачка по име на езика. Данните са в `src/data/languagesData.ts` — можеш да добавяш, местиш или трием редове свободно, всеки има `language`, `phrase` и `flag` (емоджи на знаме).

## 💻 Стартиране локално

Изисква се [Node.js](https://nodejs.org) (версия 18+).

```bash
npm install
npm run dev
```

Сайтът ще се отвори на `http://localhost:5173`.

## 🚀 Публикуване в Netlify

**Вариант 1 — през сайта на Netlify (най-лесно):**
1. Направи `npm run build` локално — ще се създаде папка `dist/`.
2. Отиди на [app.netlify.com/drop](https://app.netlify.com/drop) и провлачи папката `dist/` там.
3. Готово — получаваш линк за споделяне.

**Вариант 2 — през GitHub:**
1. Качи целия проект в GitHub хранилище.
2. В Netlify избери "Import from Git", свържи хранилището.
3. Netlify автоматично засича `netlify.toml` (build command и папка вече са зададени).
